# Coding Interview Factory — Amazon SDE II Prep

Daily Python practice for Amazon final-loop coding rounds.
Every file is paired:

- `xx_topic.py` — reference (complete code + English explanations + interview phrases)
- `xx_topic_practice.py` — skeleton with TODOs (type from scratch, repeat until automatic)

## How to use

1. Read the reference file. Read the English `# SAY:` comments OUT LOUD.
2. Open the practice file. Type from scratch — DO NOT copy/paste.
3. Run it: `python3 xx_topic_practice.py`
4. Repeat the same file until you can type it without looking.
5. Move to the next file.

## Structure

```text
phrasebook.md              ← Memorize these English phrases
01_python_basics/          ← Tools you must know cold
02_oop/                    ← Class design (Logical & Maintainable)
```
