# 🧠 Algorithms — Interview Cheat Cards

Memorize the templates. Speak the trade-offs. That's it.

## 🎯 Problem Signal → Algorithm

| Problem says... | Use this |
|---|---|
| Sorted array + pair / triplet | Two Pointers |
| Subarray / substring with constraint | Sliding Window |
| Sorted + find, "minimize / maximize X" | Binary Search |
| Shortest path unweighted | BFS |
| All paths / connected components / cycle | DFS |
| All subsets / permutations / combinations | Backtracking |
| Overlapping subproblems / counting / opt | DP |
| Local optimal leads to global | Greedy |
| Dependencies / prerequisites | Topological Sort |
| Top K / Kth largest | Heap |
| Next greater / sliding window max | Monotonic Stack/Deque |
| Many range sum queries | Prefix Sum |
