# 🎯 Interview Steps — Drill One Per Day

22 micro-files. Each one isolates a single step from the master template
([`../interview_template.md`](../interview_template.md)).

## How to use

1. **Pick one file per study session.**
2. Read the English phrases OUT LOUD 5x.
3. Memorize the knowledge table.
4. Do the drill at the bottom.
5. Move to the next file when phrases are automatic.
