# 📚 Data Structures — Interview Cheat Cards

Memorize the English. Speak it confidently. That's it.

## 🎯 Problem Signal → Data Structure

| Problem says... | Use this |
|---|---|
| Find pair / complement / duplicate | Hash Map / Set |
| Top K / Kth largest | Heap (size K) |
| Sliding window max/min | Monotonic Deque |
| Sorted input + find | Binary Search |
| Shortest path unweighted | Queue (BFS) |
| All paths / combinations | DFS / Backtracking |
| Match brackets / undo | Stack |
| Prefix lookup | Trie |
| Connected components | Union-Find |
| LRU / recency | Hash Map + Doubly Linked List |
| Schedule / dependency | Topological Sort |
