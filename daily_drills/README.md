# 🥋 Daily Drills — Type From Zero

> **Goal**: Build muscle memory. These are the patterns you MUST type cold in the interview (no IDE, no autocomplete).

## ⏰ Daily routine

```text
Every morning (15-20 min):
1. Pick 1 file (rotate through the 9)
2. Open in plain editor (no autocomplete) — Google Docs, Notepad, or VSCode without extensions
3. Type EVERY function from scratch — no peeking
4. Run python3 file.py
5. Iterate until ✅ All tests passed.

Target: each file in under 15 min by Day 7.
```

## 🚫 NO-PEEK rule

- ❌ Don't open `03_patterns/`, `leetcode_amazon/`, or any reference
- ❌ Don't use autocomplete / IntelliSense
- ❌ Don't use AI / ChatGPT
- ✅ If totally stuck for 5 min → check reference, learn, then close it and retry from scratch
